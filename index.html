Algorithms x Performance

Three indexes, one engine

Brute Force scans everything — the honest baseline. KD-Tree partitions 16-dimensional space and prunes whole branches. HNSW builds a multilayer graph and zig-zags from a sparse top layer down to a dense bottom one, hitting the same neighbors in a fraction of the steps.

Pick the metric, see the geometry

Cosine, Euclidean, and Manhattan distance reshape the same 16D space differently. Switch metrics in the UI and watch the PCA scatter plot redraw — clusters for OS, CN, DBMS, DSA, and AI/ML rearrange themselves in real time.
